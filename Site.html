QA Core Knowledge Course

General information about this course

This course aims to provide potential QA Engineers with the basic knowledge required to build a successful career in IT. It consists of 4 free online lessons that take place via Zoom Cloud Meetings platform. The total duration of the course is 12 hours.

The course program covers the following topics:
  1. Web communication basics (4 hours)
    • Client-server-database
    • HTTP Protocol
    • API/REST/JSON
    • SPA/MPA
    • Cache/Cookies/LocalStorage
    • API testing
  2. HTML/CSS/JavaScript (4 hours)
    • Introduction to HTML
    • Introduction to CSS
    • Introduction to JavaScript
    • Chrome Dev Tools/Firebug/Safari Dev Tools
    • API testing
  3. SQL (2 hours)
    • CRUD
    • Primary/Foreign keys
    • Relationships(1-1, 1-many, many-many)
    • Joins(left, right, inner, full)
  4. Git/Bash (2 hours)
    • Shell commands
    • Git commands
